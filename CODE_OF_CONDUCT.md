# Code of Conduct

Be useful, direct, and respectful. This project is for operators and agent builders working around sensitive auth flows; keep reports safe and avoid dumping private credentials or account data.

Unacceptable behavior includes harassment, doxxing, publishing secrets, intentionally misleading setup advice, or pressuring maintainers/contributors to weaken safety guardrails.

To report a problem, open a GitHub issue if it is public/community-safe. For security-sensitive reports, follow `SECURITY.md` instead.
