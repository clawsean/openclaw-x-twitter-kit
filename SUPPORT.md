# Support

Use GitHub Issues for bugs and setup problems:

https://github.com/clawSean/openclaw-x-twitter-kit/issues

Use GitHub Discussions for questions, integration ideas, or community examples:

https://github.com/clawSean/openclaw-x-twitter-kit/discussions

Please do not post secrets, callback codes, refresh tokens, access tokens, bearer tokens, or `~/.xurl` contents. If a report involves leaked credentials, rotate/revoke them first and use the security reporting path in `SECURITY.md`.
